Coupler DIY para audiopac — assets
====================================

Esta carpeta hospeda los archivos descargables del coupler artesanal:

  coupler-v1.stl       — Modelo para impresión FDM/SLA (sin relleno)
  coupler-v1.3mf       — Formato con parámetros de impresión sugeridos
  coupler-v1.step      — CAD paramétrico para modificar
  assembly.pdf         — Guía de montaje con fotos
  bom.md               — Lista de materiales y proveedores sugeridos

Cuando el modelo esté listo, reemplazar estos placeholders y activar el
feature flag SHOW_COUPLER_SECTION en src/routes/CalibrationPage.tsx.
